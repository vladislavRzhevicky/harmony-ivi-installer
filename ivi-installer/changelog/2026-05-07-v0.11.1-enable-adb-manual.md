# 0.11.1 - Enable ADB tab gets a manual

The competing `AvatrAppInstaller` ships a step-by-step manual on its
"ADB по Ethernet" screen. Without it, the operator opens our tab
and immediately runs into "where do I get these IPs?" / "what cable
do I need?" - both real questions from the field, and not obvious
even to a tech who knows ADB cold.

This patch reshapes the tab into three cards, English-only:

1. **ⓘ When to use this** - one paragraph explaining the failure
   mode that brings someone here (no USB device, no ADB toggle in
   IVI settings → firmware-locked, this routine is the dealer
   unlock).
2. **🔌 Cable & network setup** - numbered steps covering: USB-
   Ethernet adapter, OBD-II ↔ RJ45 cable + pin-out, ignition state,
   DHCP-then-static IP fallback, VPN, Avatr Service Diagnostic
   conflict, expected gateway IP.
3. **▶ Run** - the existing IP form, Discover, Enable button,
   status, and pipeline.

The tab is now a `QScrollArea` so the manual fits without squashing
the form. Pipeline moved inside the Run card to keep the visual
grouping tight (steps belong to the action that triggered them).

CTA renamed `Enable ADB over Ethernet` → `Enable ADB` - the section
header already says "over Ethernet", duplicating it on the button
is noise.
