# v0.8.8 - Install-on card polish

Three small visual fixes to the screen-category cards introduced in
0.8.6:

- **Taller cards** (min height 58px, padding bumped to 12/10) so the
  diagram + title don't feel cramped.
- **Stable border** - the border is now a constant 1px in both
  states (only the color swaps between `border` and `accent`); the
  card no longer shifts position when toggled. Background and
  border-radius are explicitly redeclared in the stylesheet
  override, so the rounded corners stay round on first toggle.
- **Cabin-shaped diagram** - `ScreensDiagram` redrawn as an
  inverted pyramid (top-down view of the cabin: driver + passenger
  up front, rear bench behind). Each chip lights up to reflect the
  card's mode. Diagram size grew from 86×14 to 72×28 to fit the new
  vertical layout.
