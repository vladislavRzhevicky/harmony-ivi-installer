# 0.14.1 - Filter dropdown popup shows full category names

The "Category" filter pill is intentionally narrow to fit alongside
"Source" and the search box. Qt sized each popup item to the parent
combo's width and elided the rest, so the menu showed
`na...on / m...g / ke...ds / br...rs` instead of the full words.

Fix: in `_labeled()` we now measure the widest item with
`QFontMetrics`, set `QComboBox.view().setMinimumWidth(widest + 32)`,
and disable elision on the popup view (`Qt.ElideNone`). The pill
keeps its compact width; the popup expands as needed.

Version: 0.14.0 → 0.14.1.
