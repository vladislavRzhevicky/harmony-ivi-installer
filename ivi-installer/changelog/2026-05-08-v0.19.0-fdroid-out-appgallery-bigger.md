# 0.19.0 - F-Droid removed; AppGallery expanded to 11 categories

Two related changes ship together:

## F-Droid removed entirely

F-Droid as a catalog source is gone. Reasons:

- Out-of-scope content: 4000+ Android apps mostly aimed at general
  consumer / privacy use cases, very few of which make sense on a
  Russian/CIS-targeted in-car head unit.
- 50 MB index download on every refresh.
- Installer-as-marketplace was confusing: AppGallery is the channel
  the head unit firmware actually expects, F-Droid was a parallel
  store.

Removed:
- `ivi_installer/sources/fdroid.py`
- `from_fdroid_index` and the `_FDROID_CATEGORY_MAP` in `catalog.py`
- `tests/test_sources_fdroid.py` and the F-Droid coverage in
  `tests/test_catalog.py`
- The `fdroid` entry in `SOURCE_DISPLAY` and in the Source filter
  dropdown
- The `Show all F-Droid` toolbar pill, the
  `fdroid_toggle_requested` signal, and the QSettings key
  `store/fdroid_enabled`
- The `include_fdroid` flag on `CatalogFetchWorker`
- The bundled `resources/store-icons/` PNGs (32 files) - they were
  fast-path icons for the F-Droid entries; AppGallery icons load
  from the AG CDN with the existing disk cache

## AppGallery: 11 categories, 12-page walk

Expanded `DEFAULT_CATEGORY_URIS` from 4 to 11 by mining the live
`internal.getTemplate` → "Categories" → "Apps" tab tree:

* top, cars, navigation, travel (existing)
* entertainment, photo_video, social, news_reading, tools,
  lifestyle, shopping (new)

Bumped `DEFAULT_MAX_PAGES` from 4 → 12 so each category can return
up to 300 items before AppGallery flips `hasNextPage=0`.

Result: a freshly-fetched index now has ~900 raw items across 11
categories, ~780 unique after dedup (was ~150 with 4 categories ×
4 pages). 24 h disk cache means subsequent launches are still
instant.

## UX

The Store tab now has a single source toggle, **"Show all
AppGallery"**, opt-in. By default the user sees the curated
~40-app extras list; toggling on triggers the live category walk
(~30 s on cold cache, milliseconds on warm). Preference persists
in `QSettings`.

## Tests

299 passed. Renamed/cleaned:
- All synthetic catalog fixtures use `appgallery` or `direct` now,
  not `fdroid`
- Toggle tests cover only the AppGallery toggle
- Source-registry test asserts exactly two kinds

Version: 0.18.0 → 0.19.0 (minor - visible UX change + a removed
source).
